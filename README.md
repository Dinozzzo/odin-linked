# Linked List (JavaScript)

This project implements a Linked List data structure in JavaScript as part of The Odin Project curriculum.

It includes common methods such as append, prepend, find, insert, remove, and traversal utilities.

## What I learned

- The difference between an array and a linked list
- How a linked list works internally without using arrays
- How to traverse nodes using references (nextNode)
- How to insert and remove elements by updating pointers
- How to handle edge cases like empty lists or invalid indexes

## Features

- Append values to the list
- Prepend values to the list
- Get the size of the list
- Access head and tail
- Find values and indexes
- Insert values at a specific index
- Remove nodes at a specific index
- Convert the list to a readable string
